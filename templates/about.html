About page

This is a real time bus tracking system for PESIT BSC which monitors buses in real time and provides students/teachers/management with information on the whereabouts of their college bus.

This web app is still in beta so expect a lot of bugs.

This web app is made in Django (Python Framework).

This is an open-source project and you can download the source code from the Github Repository.
